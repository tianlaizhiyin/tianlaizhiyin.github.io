Markdown基本语法

Markdown 是一种轻量级标记语言，让写作者专注于写作而不用关注样式。Coding 的许多版块均采用了 Markdown 语法，比如冒泡、讨论、Pull Request 等。

标题

用Markdown书写时，只需要在标题文字前加”#”来表示，每一级多一个”#”，最大支持六级标题。
示例如下：

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

效果如下：

这是一级标题

这是二级标题

这是三级标题

这是四级标题

这是五级标题
这是六级标题

字体

**加粗**
*斜体*
***斜体加粗***
~删除线~

效果如下：
加粗
斜体
斜体加粗
删除线

引用

在引用的文字前加>即可。也可以嵌套引用。
示例：

>这是引用的内容
>>二次引用的内容
>>>>>>多层引用的内容

效果如下：

这是引用的内容

二次引用的内容

多层引用的内容

分割线

三个以上的”-“或者”*”都可以。
示例：

---
----
***
****

效果如下：

可以看到，显示效果是一样的。

图片

语法如下：

![图片alt](图片地址 "图片title")
图片alt就是显示在图片下面的文字，相当于图片的解释内容。
图片title是图片的标题，当鼠标移到图片上时显示的内容。title可加可不加。

示例：

![百度Logo](https://dss0.bdstatic.com/5aV1bjqh_Q23odCf/static/superman/img/logo_top-e3b63a0b1b.png "百度")

效果如下：

超链接

语法：

![超链接名](超链接地址 "超链接title")
title可加可不加

示例：

[百度](https://baidu.com)
[追梦时光](https://blog.62young.com)

效果如下：
百度
追梦时光

列表

无序列表

用”-“”+””*”任何一种都可以。
示例：

- 列表内容
+ 列表内容
* 列表内容
注意中间的空格

效果如下：

  • 列表内容
  • 列表内容
  • 列表内容

有序列表

语法：
数字加点”.”

1. 列表内容
2. 列表内容
3. 列表内容

效果如下：

  1. 列表内容
  2. 列表内容
  3. 列表内容

列表嵌套

上一级和下一级之间留三个空格即可
示例：

* 一级无序列表内容
   * 二级无序内容
   * 二级无序内容
      * 三级无序内容
   * 二级无序内容

表格

语法：

表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略

示例：

姓名|技能|排行
--|:--:|--:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟

效果如下：
姓名|技能|排行
–|:–:|–:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟

代码

单行代码

示例：

`create database hero;`

效果如下：
create database hero;

代码块

示例：

(```javascript
    function fun(){
         echo "这是一句非常牛逼的代码";
    }
    fun();
```)

注：为了防止转译，前后三个反引号处加了小括号，实际是没有的。这里只是用来演示，实际中去掉两边小括号即可。

效果如下：

    function fun(){
         echo "这是一句非常牛逼的代码";
    }
    fun();

示例：

st=>start: begin
op=>operation: My Operation
cond=>condition: Yes or No?
e=>end
st->op->cond
cond(yes)->e
cond(no)->op

```

注：为了防止转译，flow加了小括号，实际是没有的。这里只是用来演示，实际中去掉两边小括号即可。
